== README

This is the back-end to the Blog of Many Posts.  It serves JSON to an
Ember.js front-end (called BLOG2-ember in my github).

Rails version: 4.2.0

Ruby version: 2.1.5p273


##Installation

* `git clone <repository-url>` this repository
* change into the new directory
* bundle install

##Running / Development
* rails s to start a server
* Visit the app at http://localhost:3000/posts

##Heroku

You can see this backend live at https://blog-of-many-posts.herokuapp.com/posts
